# E-Commerce App (Flask + HTML/CSS)

A complete basic e-commerce app with:
- Register new user
- Login/logout
- Product list with stock quantity
- Add to cart and update cart quantity
- Place order with delivery address
- Admin-only product management (add, delete, update price)
- Multi-language UI text switcher (English, Hindi, Spanish)
